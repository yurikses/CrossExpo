## üá¨üáß Description

**CrossExpo** is a client-side SPA (React + TypeScript) that:

1. **Accepts a word list** with clues via a convenient UI
2. **Generates a crossword** ‚Äî a greedy algorithm with 80 random restarts for best placement
3. **Displays the result** ‚Äî interactive grid with a "filled / empty" view toggle
4. **Exports to DOCX** ‚Äî a ready-made single-page A4 document with grid and clue list

Everything runs entirely in the browser ‚Äî no servers, no API keys.

### Features

| Feature | Description |
|---------|-------------|
| üìù **Two input modes** | Table mode (Word + Clue fields per row) and text mode (`word - clue` per line) |
| ‚ö° **Generation** | Algorithm places words at letter intersections, optimizing for compactness |
| üîÑ **Regenerate** | Creates a new layout variant from the same words |
| üî§/‚¨ú **View toggle** | "Filled" shows letters and answers, "Empty" shows blank grid for solving |
| üìÑ **DOCX export** | Two variants: empty puzzle and answer key; A4, auto landscape/portrait |
| üí° **Example** | Quick-load demo data for testing |
| üì± **Responsive** | Desktop, tablet, and mobile friendly |
